# Temperature-convertor
This simple temperature converter is a Python script that allows you to convert temperatures between Celsius, Fahrenheit, and Kelvin easily. Whether you're a student learning about temperature units or just need a quick way to convert temperatures, this tool is designed to be user-friendly and efficient.

Features:

Converts temperatures between Celsius, Fahrenheit, and Kelvin.
Clean and easy-to-understand Python code.
Command-line interface for quick conversions.
